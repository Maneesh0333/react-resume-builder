# 📄 ResumeForge
A modern and responsive resume builder built with React, TypeScript, React Hook Form, and Yup. Create professional resumes with live preview, dynamic sections, validation, and PDF export.


## ✨ Features

### 👤 Personal Information

* Full name and job title
* Email and phone validation
* Location details
* LinkedIn profile
* Portfolio website
* Professional summary

### 🎓 Education

* Add multiple education entries
* School and university information
* Degree and field of study
* Start and end year
* CGPA / Percentage support
* Dynamic add and remove functionality

### 💼 Work Experience

* Multiple experience entries
* Company and job title
* Employment dates
* Current job support
* Responsibilities and achievements
* Dynamic bullet points

### 🛠️ Skills

* Add technical and soft skills
* Quick-add suggestions
* Duplicate skill prevention
* Remove skills instantly
* Skill badges preview

### 👀 Live Resume Preview

* Real-time updates
* Professional resume layout
* Responsive design
* Instant preview while editing

### 📄 PDF Export

* Download resume as PDF
* Print-friendly layout
* High-quality export


## 🚀 Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS

### Form Management

* React Hook Form
* useFieldArray
* FormProvider
* useFormContext

### Validation

* Yup
* @hookform/resolvers

### Utilities

* Day.js
* Lucide React

### PDF Export

* @react-pdf/renderer
* react-to-print


## 📦 Installation

```bash
git https://github.com/Maneesh0333/react-resume-builder.git

cd react-resume-builder

npm install

npm run dev
```
